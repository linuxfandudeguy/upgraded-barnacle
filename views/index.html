Upgraded Barnacle API

Generate pixel art directly from bitmap strings. Convert 16x16 bitmap strings into vibrant images using our API.

Get Started

Why Use the Upgraded Barnacle API?

  • Generate Pixel Art: Convert 16x16 bitmap strings into pixel art images.
  • Simple & Secure: Handle requests securely with an easy-to-use API.
  • Customization: Tailor your images with unique bitmap strings for creative results.
  • Fast Processing: Quick response times with minimal latency.

Getting Started

Follow these steps to start using the Upgraded Barnacle API:

  • Install the API: Clone the repository from GitHub.
  • Send API Requests: Use the provided API endpoints to generate pixel art from your bitmap string.
  • Customize: Modify the bitmap string for different pixel art generation.

Example of a request:

GET https://upgraded-barnacle-api.example.com/generate?bitmap=0000000000000000-0000000000000000-0000000000000000...
