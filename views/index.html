Upgraded Barnacle API

Effortlessly generate bitmap images using a 16x16 grid with just a string. Bring your pixel art creations to life!

Get Started

Overview

The Upgraded Barnacle API allows you to generate bitmap images in a 16x16 grid format using the canvas library. This API converts a bitmap string into a PNG image, where each character in the bitmap represents a specific color.

Base10 Color Key

The following color codes are used in the bitmap string:

Code Color
0Black
1White
2Red
3Green
4Blue
5Yellow
6Magenta
7Cyan
8Orange
9Gray

API Endpoint

GET /generate

This endpoint generates a PNG image from a bitmap string.

Base URL

The API is hosted at: Visit the API

Query Parameters

  • bitmap (required): A string representing the 16x16 bitmap. Each line must contain 16 digits, and the lines must be separated by -.

Example Request

GET https://upgraded-barnacle-rust.vercel.app/generate?bitmap=0000000000000000-0000000000000000-0000000000000000-0000000222200000-0000002222220000-0000002277770000-0000222277770000-0000222222220000-0000222222220000-0000222222220000-0000222222220000-0000002200220000-0000002200220000-0000000000000000-0000000000000000-0000000000000000

Example Response

A PNG image is returned that visually represents the provided bitmap.

Use Cases

The Barnacle API can be used in various applications, including:

  • Pixel art generation tools
  • Games or applications with bitmap-based design elements
  • Data visualization using pixel grids

Additional Notes

The API allows you to generate 16x16 pixel bitmap images, but be mindful of the limitations related to the size and complexity of your requests. Ensure the bitmap string adheres to the 16x16 grid structure to avoid errors.
