Overview

The Upgraded Barnacle API transforms bitmap strings into colorful 16x16 PNG images, enabling pixel-perfect visualizations from a string of numbers. Ideal for developers, designers, and artists who want to quickly visualize pixel art!

API Endpoint

The API provides an easy-to-use endpoint to generate bitmap images in seconds. Simply provide the bitmap string and receive a PNG image in return.

Base URL

Visit the API

Query Parameters

  • bitmap (required): A 16x16 bitmap string, each line must contain 16 digits and the lines are separated by -.

Example Request

GET https://upgraded-barnacle-rust.vercel.app/generate?bitmap=0000000000000000-0000000000000000-0000000000000000-0000000222200000-0000002222220000-0000002277770000-0000222277770000-0000222222220000-0000222222220000-0000222222220000-0000222222220000-0000002200220000-0000002200220000-0000000000000000-0000000000000000-0000000000000000
