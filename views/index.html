Updated Barnacle API Documentation

This documentation provides all the information you need to use the Updated Barnacle API for generating bitmap images.

Overview

The Updated Barnacle API allows users to generate bitmap images using a 16x16 grid format and the canvas library. Each character in the bitmap string corresponds to a color, which is rendered into a PNG image.

Base10 Color Key

Code Color
0 Black
1 White
2 Red
3 Green
4 Blue
5 Yellow
6 Magenta
7 Cyan
8 Orange
9 Gray

API Endpoint

GET /generate

This endpoint generates a PNG image from a 16x16 bitmap string. Each bitmap line must contain 16 digits, separated by -.

Base URL

https://upgraded-barnacle-rust.vercel.app/

Example Request

GET https://upgraded-barnacle-rust.vercel.app/generate?bitmap=0000000000000000-0000000000000000-0000000000000000-0000000222200000-0000002222220000-0000002277770000-0000222277770000-0000222222220000-0000222222220000-0000222222220000-0000222222220000-0000002200220000-0000002200220000-0000000000000000-0000000000000000-0000000000000000

Example Response

Error Responses

  • 400 Bad Request: Returned if the bitmap is invalid. Example: Invalid bitmap. Must be 16 lines of 16 digits each.
  • 500 Internal Server Error: Returned if the image cannot be generated.
