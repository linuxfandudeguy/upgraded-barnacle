Updated Barnacle API Documentation

This API enables you to generate bitmap images using a simple 16x16 grid format. Perfect for pixel art or experimentation with bitmap-based graphics.

Overview

The Updated Barnacle API creates PNG images based on a 16x16 bitmap grid. Each character in the grid corresponds to a specific color, as defined by the Base10 Color Key. Any missing or empty sections of the grid will default to black.

Base10 Color Key

Code Color
0 Black
1 White
2 Red
3 Green
4 Blue
5 Yellow
6 Magenta
7 Cyan
8 Orange
9 Gray

API Endpoint

GET /generate

Generates a PNG image from a 16x16 bitmap string.

Base URL: https://upgraded-barnacle-rust.vercel.app/

Query Parameters

  • bitmap (required): A 16x16 bitmap string. Each line should have exactly 16 digits, separated by hyphens (-).

Example Request

GET https://upgraded-barnacle-rust.vercel.app/generate?bitmap=0000000000000000-0000000000000000-0000000000000000-0000000222200000-0000002222220000-0000002277770000-0000222277770000-0000222222220000-0000222222220000-0000222222220000-0000222222220000-0000002200220000-0000002200220000-0000000000000000-0000000000000000-0000000000000000

Example Response

A PNG image representing the bitmap.

Error Responses

  • 400 Bad Request: Invalid bitmap format.
  • 500 Internal Server Error: Unable to generate the image.
